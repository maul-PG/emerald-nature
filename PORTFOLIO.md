# Cara mempresentasikan proyek ini (untuk portofolio)

Panduan ringkas ini ditulis agar Anda (Rafi'i Maulana) dapat menyampaikan proyek "Emerald Nature" secara singkat dan profesional ketika bertemu perekrut atau klien.

Durasi total rekomendasi: 2—3 menit

1) Pembukaan (15–20 detik)
- Perkenalan singkat: "Saya Rafi'i Maulana — frontend developer. Ini proyek pribadi bernama Emerald Nature."
- Tujuan: showcase kemampuan frontend, desain responsif, dan penguasaan Next.js + Tailwind.

2) Visual & Live Demo (20–40 detik)
- Tunjukkan screenshot atau langsung buka Live Demo.
- Sebutkan platform deploy (mis. Vercel) dan link live demo.

3) Sorot arsitektur & fitur (40–60 detik)
- Struktur komponen: `src/components/ui` — reusable, stateless components.
- Styling: Tailwind CSS untuk konsistensi dan kecepatan pengembangan.
- Performa: optimasi gambar, build-time rendering untuk halaman statis.

4) Demo kode ringkas (30–45 detik)
- Buka satu komponen (mis. `src/components/ui/button.tsx`) dan jelaskan props serta prinsip desainnya.

5) Penutup (15–30 detik)
- Ajak perekrut/klien melihat Live Demo dan GitHub repo: https://github.com/maul-PG/emerald-nature
- Berikan kontak: maulanarafii036@gmail.com

Tips tambahan
- Siapkan 1-2 pertanyaan teknis yang mungkin ditanyakan (mis. alasan memilih App Router, optimasi performa).
- Tambahkan badge "Live" dan screenshot di README untuk memudahkan reviewer.
